\f0\fs24 \cf0 # Smart Proctoring System\
\
## Project Overview\
The Smart Proctoring System is a Computer Vision application designed to ensure integrity during online assessments. By utilizing the webcam, the system detects the presence of a student in real-time and logs alerts if the student looks away or leaves the frame. This automates the invigilation process, making it scalable and efficient.\
\
## Features\
* **Real-time Face Detection:** Utilizes OpenCV's Haar Cascade algorithms to accurately locate faces in a video feed.\
* **Live Status Monitoring:** Dynamically updates the interface to show "Student Present" (Green) or "Alert: Student Missing" (Red).\
* **Visual Feedback:** Draws bounding boxes around detected subjects for immediate visual verification.\
* **Lightweight Performance:** Optimized to run smoothly on standard laptops without requiring a GPU.\
\
## Technologies Used\
* **Language:** Python 3.x\
* **Libraries:** * `opencv-python` (Computer Vision)\
    * `numpy` (Numerical operations)\
* **Algorithm:** Haar Cascade Classifiers\
\
## Steps to Install & Run\
1.  **Clone the Repository:**\
    ```bash\
    git clone [https://github.com/yourusername/smart-proctoring-system.git](https://github.com/yourusername/smart-proctoring-system.git)\
    cd smart-proctoring-system\
    ```\
2.  **Install Dependencies:**\
    ```bash\
    pip install -r requirements.txt\
    ```\
3.  **Run the Application:**\
    ```bash\
    python main.py\
    ```\
4.  **Usage:**\
    * The webcam will open automatically.\
    * Sit in front of the camera to see the "Student Present" status.\
    * Move out of frame to test the "Alert" status.\
    * Press **'q'** to close the application.\
\
## Testing Instructions\
* **Positive Case:** Ensure the user's face is well-lit and facing the camera. The system should draw a green box.\
* **Negative Case:** Cover the camera or move out of the frame. The system should display a red "Alert" text.}
